Rhythm Game - Piano Tiles on the DE1-SoC

A Rhythm Game similar to Piano Tiles made in the C language for the DE1-SoC Board.

Wrote low-level procedural code with strict timing constraints, where time complexity and efficiency was required for functionality. Designed multiple real-time parallel modules for audio output, game events, and integrated them with peripheral input signals.

StuGIS - Usable and Responsive Mapping + Navigation

A GIS desktop app designed to make student life easier, coded in C++ for the Linux system.

From displaying and filtering key points, providing navigation, to finding an optimized path hitting many destinations, StuGIS can do it all. My team and I worked on navigation algorithms, Solved a constrained Traveling Salesman Problem, and developed the UI.
